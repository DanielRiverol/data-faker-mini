# Módulo: UUID 🆔

Generador de identificadores únicos universales.

## Métodos

### `v4()`
Genera un UUID versión 4 estándar (`xxxxxxxx-xxxx-4xxx-yxxx-xxxxxxxxxxxx`).

```javascript
faker.uuid.v4() // "e4f8a3b2-7d1c-4e9f-8a2b-1c3d4e5f6a7b"
```
::: tip Recomendación
Ideal para usar como clave primaria (Primary Key) en tus datasets de prueba.
:::
