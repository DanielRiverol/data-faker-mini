# Módulo: Location 📍

Datos geográficos basados en el locale seleccionado.

## Métodos

### `city()`
Devuelve una ciudad importante del país configurado en el `locale`.

```javascript
// Con locale: "es_AR"
faker.location.city() // "Mar del Plata"

// Con locale: "fr_FR"
faker.location.city() // "Marseille"
```
### `country()`

Devuelve el nombre del país del locale actual.
```javascript

faker.location.country() // "Argentina"
```
